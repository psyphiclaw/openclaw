# 数据清洗规则

## 各模态清洗规则

### FaceReader
- 人脸检测结果为 0 的帧：标记为缺失，不删除
- AU 值为 0 时需区分"未检测到"和"实际为 0"
- VAD 维度突然跳变（相邻帧差 > 0.5）：标记为可疑

### EEG
- 幅值超过 ±100μV：标记为伪迹
- 平坦段（方差 < 1e-6）：标记为断连
- 50/60Hz 工频干扰： notch 滤波
- EOG/EMG 伪迹：ICA 去除

### 眼动
- 注视丢失（validity ≠ 0）：删除或插值
- 瞳孔直径为 0 或负值：标记为眨眼
- 瞳孔直径突然跳变（>3SD）：线性插值

### 生理信号
- ECG：基线漂移 > 0.5mV → 高通滤波 0.5Hz
- EDA：运动伪迹 → 低通滤波 5Hz
- EMG：工频干扰 → notch 滤波

## 标准化方法选择
| 场景 | 方法 |
|------|------|
| 被试间比较 | Z-score |
| 条件内对比 | 基线校正 |
| 多指标融合 | Min-Max |
| 非正态分布 | 百分位秩 |

## 质量控制检查清单
- [ ] 采样率是否一致
- [ ] 时间戳是否连续
- [ ] 缺失值比例 < 10%
- [ ] 信号幅值在合理范围
- [ ] 无重复时间戳
