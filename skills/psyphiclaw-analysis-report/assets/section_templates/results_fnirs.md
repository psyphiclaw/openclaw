# fNIRS 分析结果

## 写作指引

在撰写 fNIRS 分析结果时，请包含以下内容：

### 1. 数据质量
- 信号质量评分（Scalp Coupling Index）
- 排除的通道比例及原因
- 被试排除标准及数量

### 2. 血红蛋白浓度变化
- HbO 和 HbR 的时间过程描述
- 感兴趣区域的激活模式
- 与基线的比较结果

### 3. 统计结果
- GLM 分析的统计参数
- 显著激活的通道（FDR 校正后）
- 效应量报告

### 4. 空间分布
- 激活的空间模式描述
- 与已知脑区的对应关系
- 左右半球差异

### 模板

```
fNIRS 数据分析结果显示，在 [任务条件] 下，[脑区] 区域的氧合血红蛋白 (HbO) 
浓度显著增加（M = [X.XX] μM, SD = [X.XX], p < [.XXX], d = [X.XX]）。
[HbR] 变化呈相反趋势（M = [X.XX] μM, SD = [X.XX], p < [.XXX]）。
该激活模式与先前 [文献] 的研究结果一致，提示 [认知过程] 的参与。
```
