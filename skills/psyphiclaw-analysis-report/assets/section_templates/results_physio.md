# 生理信号分析结果

## 报告内容
- ECG：心率、心率变异性（HRV）、RMSSD
- EDA/GSR：皮肤电导水平（SCL）、皮肤电导响应（SCR）
- EMG：肌电活动（如 zygomaticus, corrugator）
- 呼吸：呼吸频率、呼吸变异性
- PPG：血氧、脉搏波

## 统计报告规范
- 明确时间窗口和基线校正方法
- 标注信号单位和采样率
