# 讨论 / Discussion

## 写作目标
解释结果的意义，与已有文献对比，承认局限性。

## 内容要点
1. **主要发现总结**：用通俗语言重述核心结果
2. **与文献对比**：支持和 contradicts 已有发现
3. **理论意义**：对相关理论框架的贡献
4. **实践意义**：对实际应用的启示
5. **局限性**：样本量、技术限制、方法论问题
6. **未来方向**：改进和扩展建议

## 写作模板
```
本研究的主要发现是 [一句话总结]。这一结果与 [作者, 年份] 的发现 [一致/不一致]，
可能的原因包括 [解释]。然而，本研究存在以下局限：[列举]。
未来研究可考虑 [改进建议]。
```
