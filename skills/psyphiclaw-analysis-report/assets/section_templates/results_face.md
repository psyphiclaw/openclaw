# 表情分析结果 / Facial Expression Results

## 报告内容
- Action Unit 激活强度（均值 ± 标准差）
- 基本情绪分类的时间进程
- 组间/条件间差异的统计检验结果
- 关键时间窗口识别

使用 APA 格式：*F*(1, 28) = 5.32, *p* = .028, η²p = .16
