# 实验概述 / Overview

## 写作目标
清晰描述研究背景、目的和实验设计，让读者快速理解研究的核心问题。

## 内容要点
1. **研究背景**：为什么做这个实验？已有文献的不足是什么？
2. **研究目的**：核心假设和预期结果
3. **实验设计**：被试数量、组别划分、实验任务流程
4. **数据采集**：使用了哪些模态（表情、EEG、眼动、生理信号、fNIRS）
5. **伦理声明**：伦理审批编号

## 写作模板
```
本研究旨在探讨 [研究问题]。共招募 [N] 名被试，采用 [实验范式] 设计。
实验采集了 [模态列表] 数据。主要假设为 [H1/H2/...]。
```

## 字数建议
中文 300-500 字 / English 200-350 words
