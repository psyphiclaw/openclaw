# 方法 / Methods

## 写作目标
提供足够的实验细节，使研究可重复。

## 内容要点
1. **被试信息**：人口统计学特征、纳入/排除标准
2. **实验材料与刺激**：使用的刺激材料、设备
3. **实验流程**：时间线（准备 → 基线 → 任务 → 恢复）
4. **数据采集参数**：
   - 表情：采样率、面部模型、检测区域
   - EEG：电极分布、参考电极、滤波参数
   - 眼动：采样率、追踪模式、校准方法
   - 生理信号：通道列表、采样率、传感器型号
   - fNIRS：通道数、波长、光源分布
5. **数据预处理**：各模态的清洗与转换步骤
6. **统计分析方法**：使用的统计检验和模型

## 注意
- 读者应能根据此描述复现实验
- 明确标注软件版本号和参数设置
