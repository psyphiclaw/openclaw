# 报告各部分内容规范

## 统计结果报告规范

### 必要元素
- 描述性统计：均值 (M)、标准差 (SD)
- 推断性统计：检验统计量 (t/F/χ²)、自由度、p 值
- 效应量：Cohen's d, η², 或 r
- 置信区间（推荐 95% CI）

### 格式示例（APA）
```
t(28) = 3.45, p = .002, d = 0.65, 95% CI [0.23, 1.07]
F(2, 57) = 8.12, p = .001, η² = 0.22
r(30) = .45, p = .012, 95% CI [.10, .71]
```
