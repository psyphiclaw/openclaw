# 结果解释注意事项

## 🔴 因果推断警告
- 相关性 ≠ 因果性
- 仅表述为"关联"/"相关"，避免使用"导致"/"引起"
- 实验设计为因果推断时方可使用因果语言

## 🔴 过度解读预防
- p < .05 不等于"重大发现"
- 必须报告效应量
- 小样本研究的结论应更谨慎
- 探索性分析需要明确标注

## 🔴 多重比较
- 多次检验时必须进行校正（FDR/Bonferroni）
- 报告校正后的 p 值
- 未经校正的结果应标注为"探索性"

## 🔴 多模态融合解释
- 跨模态关联不等于机制性联系
- 时间上的接近不等于因果关系
- 应考虑混杂变量（运动伪迹、环境噪声）
- 建议结合已有理论框架进行解释
