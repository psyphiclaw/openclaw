# 分析方法参考 (Analysis Methods Reference)

## 1. 相关性分析

### Pearson vs Spearman
| | Pearson | Spearman |
|---|---------|----------|
| 假设 | 线性关系，正态分布 | 单调关系，无分布假设 |
| 适用 | 连续变量，近似正态 | 有序变量，非正态/离群值 |
| 灵敏度 | 对离群值敏感 | 鲁棒 |

### 滑动窗口相关
- **窗口大小**: 至少包含 50-100 个数据点
- **步长**: 通常为窗口大小的 10-25%
- **EEG (250Hz)**: 推荐 2-5s 窗口, 0.5s 步长
- **表情 (30Hz)**: 推荐 5-10s 窗口, 1s 步长

### 置换检验
- **推荐次数**: ≥ 1000（报告用 ≥ 5000）
- **seed**: 固定随机种子确保可重复性
- **双侧检验**: 使用 |r_perm| ≥ |r_obs|

## 2. 时间锁相分析

### ERP 分析
- **基线校正**: 通常用刺激前 200ms
- **低通滤波**: 30-40Hz（经典 ERP）或 20Hz（晚成分）
- **伪迹剔除**: ±100μV 阈值

### 跨模态时间锁相
- **表情数据**: 锁相到刺激，观察情绪变化的时间进程
- **生理信号**: EDA 响应延迟 1-3s, HRV 变化 2-5s
- **注意**: 不同模态的响应延迟不同！

### 统计比较
- **时间窗口**: 基于先验假设（非数据驱动）
- **多重比较**: 时间点 × 电极 → 需要聚类校正或 FDR

## 3. 回归分析

### OLS 假设检验清单
1. ✅ 线性关系（散点图检查）
2. ✅ 残差正态性（Shapiro-Wilk 检验）
3. ✅ 同方差性（Breusch-Pagan 检验）
4. ✅ 无多重共线性（VIF < 5）
5. ✅ 无自相关（Durbin-Watson ≈ 2）

### 混合效应模型
- **何时使用**: 重复测量、被试内设计、嵌套数据
- **随机效应**: 被试 (subject)、项目 (item)
- **最大模型**: 随机截距 + 随机斜率
- **REML vs ML**: 估计方差分量用 REML，模型比较用 ML

### 多重比较校正
| 方法 | 适用场景 | 特点 |
|------|----------|------|
| Bonferroni | 少量比较 (<10) | 保守，控制 FWER |
| FDR (BH) | 中等数量比较 | 控制错误发现率，更有效 |
| 基于聚类 | 时间×空间 ERP | 考虑相邻点相关性 |

## 4. 效应量报告

### Cohen's d 参考值
- 0.2 = 小效应
- 0.5 = 中效应
- 0.8 = 大效应

### R² 参考值（行为/生理数据）
- 0.02 = 小效应
- 0.13 = 中效应
- 0.26 = 大效应

### 报告格式
> "EEG alpha power significantly predicted FaceReader valence ratings, β = 0.34, SE = 0.08, p = .002 (FDR-corrected), R² = 0.12."

---

## 5. 常见陷阱

1. **伪相关**: 时间序列的 autocorrelation 导致的虚假相关 → 使用滑动窗口或 ARIMA 预白化
2. **采样率不匹配**: 直接比较不同采样率的数据 → 先重采样
3. **选择偏差**: 只报告显著的通道/时间窗口 → 预注册分析计划
4. **循环分析**: 用同一数据选择 ROI 再检验 → 交叉验证或独立样本
5. **多重比较爆炸**: EEG (64 ch × 500 timepoints) → 必须校正
